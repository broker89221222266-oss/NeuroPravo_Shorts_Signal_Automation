# Manual Import Workflow

Цель: быстро собрать 20-50 коротких роликов, которые уже получили сильную реакцию, и превратить их в редакторский shortlist для сценариев НейроПраво.

Инструмент ничего не публикует, не рендерит и не подключается к аккаунтам. Александр вручную выбирает, какие идеи брать дальше.

## Где брать ролики

Начните с источников, где уже видно социальное доказательство: YouTube Shorts, TikTok, Instagram Reels, VK Clips, Дзен, ViewStats, Apify export или ручные ссылки из ленты. На первой живой партии не нужно искать идеально: важнее собрать 20-50 кандидатов, чтобы проверить качество отбора.

Ищите не только юристов. Лучше работают ролики с болью: деньги, долг, договоренность, ремонт, застройщик, маркетплейс, клиент не платит, спор в семье, наследство, алименты, кассовый разрыв, ошибка в переписке, обещание на словах.

## Какие поля заполнять

Файл партии кладите в `data/manual_batches/`.

Обязательные колонки такие же, как в `data/templates/import_template.csv`:

```text
source_url,platform,published_at,author,title_or_caption,views,likes,comments,shares,saves,duration_sec,topic_hint,notes
```

Что важно заполнить обязательно:

- `source_url` - ссылка на ролик;
- `platform` - одна из разрешенных платформ;
- `published_at` - дата в формате `YYYY-MM-DD`;
- `title_or_caption` - заголовок, подпись или краткое описание ролика;
- `views` - просмотры, если видны;
- `topic_hint` - ваша короткая пометка: долг, договор, застройщик, дебиторка, маркетплейс.

Что можно оставить пустым:

- `author`, если не нужен;
- `likes`, `comments`, `shares`, `saves`, если платформа не показывает;
- `duration_sec`, если не видно;
- `notes`, если нечего добавить.

Пустые числа считаются нулем. Можно писать `12k`, `1.5m`, `12к`, `1.5м`.

## Как запускать анализ

Пример для новой партии:


Для sample batch:

```powershell
python .\scripts\generate_cards.py --input data\manual_batches\sample_manual_batch.csv --out output\sample_manual_batch --min-score 70
```

## Как читать results

Сначала открывайте:

```text
output\...\batch_summary.md
```

Там есть:

- качество партии;
- сколько платформ и тем;
- сколько старше 30 дней;
- сколько без просмотров или реакций;
- причины отсева;
- блок `Смотреть первыми`;
- топ-10 по `final_score`;
- топ-5 по `neuropravo_fit_score`.

Потом открывайте:

```text
output\...\scenario_cards.md
```

Там уже карточки сценариев: hook, механика, почему не копирование, что изменить под НейроПраво, юридически безопасная граница и компактный сценарий.

## Как Александр выбирает идеи дальше

1. Открыть `batch_summary.md`.
2. Посмотреть блок `Смотреть первыми`.
3. Выбрать 3-5 кандидатов со статусом `БРАТЬ` или `ПОДУМАТЬ`.
4. Открыть эти карточки в `scenario_cards.md`.
5. Решить: читать как есть, просить усилить сценарий или отложить тему.

Контрольная мысль: мы не копируем ролик. Мы берем сигнал рынка: какая боль, конфликт и форма удержали внимание. Текст и вывод пересобираются заново под НейроПраво.
